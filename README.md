# 超详细MIT线性代数公开课笔记-完整版

本仓库提供了一份超详细的MIT线性代数公开课笔记，该笔记由麻省理工学院的Gilbert Strang教授主讲。这份笔记涵盖了线性代数的核心概念和重要知识点，适合所有对线性代数感兴趣的学习者使用。

这份笔记详细记录了Gilbert Strang教授在MIT线性代数公开课中的讲解内容，包括但不限于以下主题：

- 向量与矩阵的基本概念

- 行列式与特征值
- 向量空间与线性变换
- 正交性与最小二乘法
- 特征值与特征向量的应用
